จัดการเว็บ
☁️ ตั้งค่า Cloud Sync (Firebase)
เชื่อมต่อ Firebase เพื่อให้ข้อมูลซิงค์ข้ามทุกบราวเซอร์และทุกเครื่องแบบ Realtime
☑️ วิธีสร้าง Firebase ฟรี (5 นาที)
1. ไปที่ console.firebase.google.com
2. กด Add project → ตั้งชื่อ → Create
3. เลือก Build → Realtime Database → Create database
4. เลือก Start in test mode → Enable
5. ไปที่ Project Settings ⚙ → General
6. กด </> Add app → ลงทะเบียน
7. Copy databaseURL มาใส่ด้านล่างนี้
🔗 เชื่อมต่อ Firebase
วาง Database URL จาก Firebase Console (รูปแบบ: https://xxx-default-rtdb.xxx.firebaseio.com)
☑️ เชื่อมต่อแล้ว
⚡ Auto Sync
เมื่อเปิด — ข้อมูลจะ push ขึ้น Cloud ทันทีทุกครั้งที่กด บันทึก
⚠ หมายเหตุ
• Firebase Realtime DB ฟรี 1GB ข้อมูล / 10GB transfer/เดือน
• ข้อมูลจะซิงค์ตัวคุณเท่านั้น (ไม่แชร์ให้คนอื่นอัตโนมัติ)
• ควรเปลี่ยน Database Rules ก่อน deploy จริง
